# Login Page Images

এই folder-এ নিচের image save করতে হবে:

## Required Image:

### 1. Doctor Image (Main Hero Image)
**File Name:** `doctor.jpg`
**Location:** `/images/login/doctor.jpg`
**Description:** 
- Smiling female doctor
- Wearing white lab coat
- Holding stethoscope (left hand holding earpieces to ears, right hand holding chest piece)
- Looking directly at camera with friendly expression
- Professional, high-quality photo
- Background should be clean or can be removed (transparent/white background preferred)

**Size:** 400x600px বা তার বেশি (portrait orientation)
**Format:** JPG/PNG
**Quality:** High resolution, professional photography

## Image Guidelines:
- Doctor should be the main focus
- Professional medical setting
- Friendly and approachable expression
- High quality and clear
- Good lighting
- Can be on white/light background or transparent

## Fallback:
যদি image load না হয়, automatically placeholder SVG show করবে।
